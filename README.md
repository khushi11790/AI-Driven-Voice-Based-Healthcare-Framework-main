# 🏥 AI-Driven Voice-Based Healthcare Framework

## 🚀 Overview

This project is an AI-powered healthcare assistant that allows users to interact through voice, analyze symptoms, and generate medical reports automatically.

## ✨ Features

* 🎙️ Voice-based interaction
* 🤖 AI-powered medical suggestions
* 📄 Automated report generation
* 🔐 Secure authentication (Clerk)
* 📊 Dashboard for session history

## 🛠️ Tech Stack

* Frontend: Next.js, React, Tailwind CSS
* Backend: Node.js
* Database: PostgreSQL (Neon)
* Authentication: Clerk
* AI Integration: OpenRouter
* Voice: Vapi

## 🔧 Setup Instructions

1. Clone the repository
2. Install dependencies:
   npm install
3. Create `.env` file using `.env.example`
4. Run the project:
   npm run dev
